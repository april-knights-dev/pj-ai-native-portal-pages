AI Native Development Project

このポータルでは、プロジェクトのテーマ、週次活動、評価推移、成長の兆しをひとつの画面で確認できます。

メンバー別 Claude Code サマリー

Setup

使用状況トラッキングの導入

各メンバーの Claude Code セッション終了時に使用状況（トークン・スキル・サブエージェント）を自動収集します。
以下のコマンドをメンバーの PC で 1 回だけ実行してください。Python 3 が必要です。

1
Stop hook をインストール（OS を選んで実行）

Mac / Linux（ターミナル）

curl -fsSL https://april-knights-dev.github.io/pj-ai-native-portal-pages/setup-claude-hook.sh | bash

Windows（PowerShell）

iex (iwr 'https://april-knights-dev.github.io/pj-ai-native-portal-pages/setup-claude-hook.ps1' -UseBasicParsing).Content

実行すると ~/.claude/settings.json に Stop hook が自動追加されます。

2
動作確認

ai-native-team-a/b/c リポジトリで Claude Code セッションを終了すると、次の更新でこの画面にデータが反映されます。

収集対象: ai-native を含むプロジェクトのみ（個人プロジェクトは収集されません）
収集内容: トークン数・使用モデル・スキル呼び出し・サブエージェント呼び出し（コードの内容は収集しません）
